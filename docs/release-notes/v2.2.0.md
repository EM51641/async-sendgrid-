# Release Notes v2.2.0

## 馃殌 New Features

### Automatic Retry with Exponential Backoff
- Built-in retry support for transient failures (429, 5xx, timeouts) via `httpx-retries`
- Configurable through `ConnectionPool` with `retry_attempts`, `backoff_factor`, and `backoff_jitter`
- Defaults to 5 retries with exponential backoff
- Per-request overrides via `sendgrid.send(email, retry=3, backoff=1.0)`
- Disable retries entirely with `retry_attempts=0`

### Python 3.14 Support
- Added official Python 3.14 support
