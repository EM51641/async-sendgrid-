# Release Notes v2.2.1

## 🐛 Bug Fixes

### Connection pool limits no longer overwritten by the transport layer
- Pool limits (`max_connections`, `max_keepalive_connections`, `keepalive_expiry`) are now correctly propagated to the underlying transport

### Per-request retry/backoff overrides are now validated
- `retry` and `backoff` overrides passed to `send()` are now validated using the same rules as `ConnectionPool` initialization

## 🔧 Improvements

### Dynamic version from package metadata
- `__version__` is now read from installed package metadata instead of being hardcoded
